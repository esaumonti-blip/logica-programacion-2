# logica-programacion-2
<p>Crear un programa en Javascript que realice lo siguiente:</p>
  <ol>
    <li>Debe solicitar al usuario la temperatura en grados Celsius por prompt o por un input.</li>
    <li>Debe convertir la temperatura ingresada de grados Celsius a grados Fahrenheit y Kelvin</li>
    <li>Debe imprimir ambos resultados por consola o por el DOM.</li>
    <li>Debe ser capaz de identificar si los datos de entrada sean de tipo number, en caso contrario debe mandar un mensaje de error y volver a solicitar los datos.</li>
    <li>Prueba tu programa con las siguientes entradas para asegurarte que funcione correctamente:</li>
    <li>
    Entrada: 
    45.
    </li>
    <li>
    Salida
    Grados Kelvin: 318.15
    Grados Fahrenheit: 113. 
    </li>
    <li>
    Entrada:
    14.
    </li>
    <li>
    Salida
    Grados Kelvin: 287.15
    Grados Fahrenheit: 57.2. 
    </li>
    <li>Pruébalo con los valores numéricos que se te ocurran.</li>
    <li>Al final de tu práctica, tienes que subir el ejercicio a tu repositorio de GitHub.</li>
    <li>Colócalo en un repositorio llamado “logica-programacion-2”</li>
  </ol>
